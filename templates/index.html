PhishShield

Explainable Phishing Detection System

About the Project

PhishShield is an academic cybersecurity project developed to identify potentially malicious links using rule-based, explainable analysis. The system focuses on transparency rather than black-box machine learning.

How PhishShield Works

  • Checks HTTPS encryption
  • Detects IP-based URLs
  • Identifies suspicious keywords
  • Analyzes domain age using WHOIS data
  • Generates a deterministic risk score
